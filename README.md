# Rejseplanen 2 for Home Assistant

Viser afgange fra danske stoppesteder via Rejseplanen API 2.0.
(HA's indbyggede rejseplanen-integration døde, da API 1.0 lukkede i december 2024.)

## Krav
Gratis API-nøgle (accessId) fra Rejseplanen Labs: https://labs.rejseplanen.dk

## Installation (HACS)
1. HACS → Custom repositories → tilføj denne repo-URL, kategori "Integration"
2. Installér, genstart Home Assistant
3. Indstillinger → Enheder & tjenester → Tilføj integration → "Rejseplanen 2"

## Indstillinger
Klik "Konfigurer" på integrationen: opdateringsinterval og linjefilter.
